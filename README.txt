Requirements: 
 - PC Platform: Linux system with python 2.7 with bluetooth support
 - Smartphone android 5.* with bluetooth support
 
Setup PC: 
	install python modules: pycrypto, PyBluez, qrcode and Pillow 
	run: python main_blue.py
	add files to the folder named "smartcipher" only after starting a session
	
Setup Android: 
	install smartcipher.apk
	in the app:
	- configure the public key (using qrcode reader) 
	- select the correct paired bluetooth device 
	
Start your session by pressing the "Start" button in the application

Enjoy :)
